# 我们的承诺 #

我们郑重承诺，所有经启信科技翻译编校的文稿的英语语言质量，均符合英文期刊发表标准。此承诺适用于所有经我们编校的文稿，与服务，返稿时间及费用无关！

如果您的文稿因为英语语言质量问题收到期刊负评，我们将无限次免费重新编校您的稿件，只要您已妥善处理了编辑的所有注释与评论并未在投稿前对已编校完成的稿件做显著修订。

我们深知文件泄密带来的严重后果，因此我们结合行业经验与科技论文、文献发表的实际需求，制定了严格的保密体系，确保客户的文件信息安全。

我们采用以下信息保密措施：

# 客户在交付尚未发表论文之前，我们会要求客户先删除文章的关键数据、图表，甚至重要结论

# 对于排版和修图服务，我们会要求客户先将关键数据以符号代替，编排工作也会交由其他非相关专业的编校人员完成

# 对于所有译者和编校人员我们都会签订保密协议，明确保密义务和保密责任

# 所有客户都可以与我司签订保密协议，明确保密义务和保密责任

# 我们承诺，您的文件的任何部分都不会泄露给第三方
