# CrediteScitech
The website for a [translation company](http://qixin-trans.com "启信翻译").

Using:
- Python/Flask
- MySQL
- Bootstrap
- jQuery
- Gunicorn
- Supervior

## Environment variables required
Set them before run this project:

`SECRET_KEY`

`DATABASE_URL`

`FLASK_CONFIG`

`CREDITE_UPLOAD_FILES_PATH`

`MAIL_SERVER`

`MAIL_PORT`

`MAIL_USE_TLS`

`MAIL_USE_SSL`

`MAIL_USERNAME`

`MAIL_PASSWORD`
